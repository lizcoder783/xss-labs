# Usage for XAMPP user

2. Now right-click and Open in Terminal
3. git clone https://github.com/lizcoder783/xss-labs.git

1. You need to goto the directory of var/www/html
2. Now right-click and Open in Terminal
3. git clone https://github.com/lizcoder783/xss-labs.git
